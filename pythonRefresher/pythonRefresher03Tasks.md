## PROBLEM 1: Age Range 
**File name:** `problem_3_01.py`

## PROBLEM 6: Movie Theater Pricing
**File name:** `problem_3_06.py`

- Children (under 13): $8

## PROBLEM 7: Triangle Validator
**File name:** `problem_3_07.py`

Ask the user for three side lengths of a triangle (a, b, c).
A valid triangle must satisfy ALL of these conditions:
- a + b > c
- a + c > b
- b + c > a

If all three conditions are true, print "Valid triangle!"
Otherwise, print "Not a valid triangle."

## PROBLEM 1: Basic Variables 
**File name:** `problem_1_01.py`

Create three variables:
- `first_name` (your first name as a string)
- `last_name` (your last name as a string)
- `age` (your age as an integer)

## PROBLEM 3: User Input 
**File name:** `problem_1_03.py`

## PROBLEM 4: Input with Numbers 
**File name:** `problem_1_04.py`

Ask the user how many hours they slept last night.

Calculate how many minutes that is (hours * 60).

## PROBLEM 5: Price Calculator
**File name:** `problem_1_05.py`
